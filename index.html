Quiet Focus — Privacy Policy

Last updated: July 15, 2026

Quiet Focus — Floating Timer is a browser extension by FrootyStudios that provides one persistent focus countdown and an optional floating timer on websites selected by the user. This policy explains what information the extension handles and how it is protected.

In short: Quiet Focus does not collect or transmit personal information, browsing history, website content, timer data, or website-display preferences. The extension contains no analytics, advertising, tracking, or remote code.

Information Quiet Focus handles

Quiet Focus handles only the information necessary to provide its timer and user-selected website display behavior:

  • The selected focus duration and timer runtime state
  • Whether the floating timer is minimized or expanded
  • Whether automatic display on HTTP and HTTPS websites is enabled
  • Website origins where the user has enabled or hidden the floating timer

The extension uses a website's origin, such as https://example.com, only to register or exclude the floating timer according to the user's choice. It does not inspect or record the content of pages, form entries, search queries, or the user's activity within a website.

Storage and Chrome Sync

Active timer state is stored in chrome.storage.local so the countdown remains accurate after Chrome suspends the extension, the browser restarts, or the computer sleeps.

Preferences—including the focus duration, automatic-display setting, enabled website origins, and hidden website exclusions—are stored with chrome.storage.sync. Chrome may synchronize these preferences through the user's signed-in Chrome profile under Google's Chrome Sync terms. FrootyStudios does not receive or maintain an external copy of this information.

Permissions and why they are used

  • activeTab — identify the active website and display the timer there after the user selects Show.
  • alarms — reliably finish an active countdown when Chrome suspends the background service worker.
  • notifications — notify the user when a focus session finishes.
  • scripting — add, remove, and restore the packaged floating timer on websites selected by the user.
  • storage — save timer state and the preferences described above.
  • Optional HTTP and HTTPS website access — display the floating timer on a specific website chosen by the user or on all websites when the user explicitly enables Show on all websites.

Website access is optional. Quiet Focus does not use it to read, collect, or transmit page content. The extension does not request access to cookies, passwords, authentication tokens, downloads, bookmarks, clipboard contents, or personal communications.

Collection, transmission, and sale

Quiet Focus does not transmit timer state, preferences, website origins, browsing activity, or page content to FrootyStudios or any analytics, advertising, tracking, or data-broker service. FrootyStudios does not sell Quiet Focus user data or use it for creditworthiness, lending, advertising, or unrelated purposes.

Remote code and third-party services

All executable code, interface assets, and fonts used by Quiet Focus are packaged with the extension. The extension does not download or execute remote code and does not load third-party analytics, advertising, tracking scripts, or external fonts.

Chrome Sync and the optional uninstall feedback form are Google services and are subject to Google's applicable privacy terms.

Optional uninstall feedback

When Quiet Focus is uninstalled, Chrome may open an optional feedback form hosted by Google Forms at this address. The extension does not append timer state, website origins, identifiers, or other extension data to the form URL.

You may close the page without submitting anything. If you choose to submit a response, the information you enter is provided to FrootyStudios through Google Forms and processed by Google under its applicable privacy terms. Voluntary feedback is used only to understand issues and improve Quiet Focus.

Data retention and deletion

Quiet Focus does not maintain an external database of timer data or preferences. Disabling or uninstalling the extension stops its local processing. Chrome removes extension-local data on uninstall; synced extension settings may remain subject to Chrome Sync behavior and the user's Google account settings.

To request deletion of feedback voluntarily submitted through the uninstall form, contact FrootyStudios using the address below.

Children's privacy

Quiet Focus is a general-purpose productivity utility and is not directed to children. FrootyStudios does not knowingly use Quiet Focus to collect personal information from children.

Changes to this policy

If Quiet Focus's data practices or this policy change, the updated policy will be published on this page with a revised Last updated date. Material changes will be disclosed as required by applicable Chrome Web Store policies.
